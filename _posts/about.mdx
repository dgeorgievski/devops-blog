# About

I always had the urge but never really had time in the past for blogging.
The breaking point came when I wanted to demo the benefits of IPFS for hosting
so called static Web sites, and their advantages over costly and difficul to manage
applications like WordPress.

There is really nothing static about them. They could be sophisticated JS
applications running in your browser. [Unhosted](http://unhosted.org) would better
describe them.

With the emergence of trully distributed technologies like IPFS, P2P, Blockchains,
Distributed IDs, P2P it is possible now
